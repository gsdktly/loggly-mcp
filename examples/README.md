# Examples

These files are **tool argument payloads** you can pass to the MCP tools.

## Files

- `connection_test.json` → `connection_test`
- `iterate_events_page.json` → `iterate_events_page`
- `field_facets.json` → `field_facets`

## Notes

- Values are safe defaults. Customize the query/time window to match your Loggly data.
- Tool arguments are sent directly to the MCP tool input schema.
